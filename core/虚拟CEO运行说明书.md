## 一、系统是什么

虚拟CEO = AI Agent作为经营总指挥，锋哥是CEO。

一句话：**从全年经营目标出发，拆到月→周→天，AI负责提醒、追踪、汇总、记账，锋哥只管执行和决策。**

核心理念：
- 全年70+经营节点自动提醒，到什么月份做什么事
- 收支双写（Notion粗账 + 飞书细账），零漏记
- 学生缴费全追踪，续费不漏人
- 周报/月报自动推送，数字透明
- 招生漏斗、家长沟通、老师管理逐步自动化

## 四、已落地的功能

### ✅ 4.1 日常记账（双写）

**使用方式：** 直接发消息给AI，如"买菜103"、"交电费300"、"黄老师工资4000"

**执行流程：**
1. AI自动识别分类（六象限）
2. 同时写入Notion支出库 + 飞书记账表
3. 返回确认：项目/金额/分类/日期
4. 个人消费自动排除

**分类规则速查：**
| 你说 | AI归为 |
|------|--------|
| 买菜/买肉/买水果/厨房 | 厨房 |
| 工资/发工资/老师工资 | 工资 |
| 房租/水电/燃气/物业 | 房租 |
| 纸/水杯/笔/垃圾袋/日用品 | 杂费 |
| 礼品/宣传单/展板/地推 | 营销 |
| 买桌子/买空调/装修 | 折旧 |

### ✅ 4.2 学生缴费管理

**使用方式：** 告诉AI缴费情况，如"刘明按月缴费交了5月的"、"张三没交期费"

**执行流程：**
1. AI更新飞书缴费管理表对应记录
2. 自动计算待缴金额和缴费状态
3. 标记未缴费人员

## 九、使用指南

### 日常记账
直接发消息即可，如：
- "买菜103"
- "交水电费300"
- "发潘老师4200"
- "买宣传单50"

- "本月花了多少" → AI从飞书拉取当月数据汇总
- "厨房这个月多少" → 按分类查
- "谁还没交费" → 查缴费表

- "今天巡查记录：XXX" → AI记录到日志
- "跟廖一凡爸爸打电话了，态度还行" → 更新PEOPLE跟进

- AI会按设定频率推送周报/月报/缴费提醒
- 全年经营节点到期自动提醒该做什么
