### 主题1：语文整体规划（小学+初中路径）

**涉及文件：** 214855、215859、220901、221904、222908、223911、224914、225917、230921、230105

**核心方法论：** 周一到周五抓积累，周末抓精读和仿写；小学抓课内本质能力，初中直接搞真题。

**具体做法：**

1. **周一到周五语文任务（按优先级）：**
   - 课内好词积累（名词、形容词、动词，尤其形容词和动词最重要）
   - 每天一个成语（用漫画成语，有图有释义有反义词，一篇1-2分钟）
   - 丁立梅修辞积累（可放周末做，因为有录好的课程）

2. **周末语文任务：**
   - 精读（用五维御学法或教辅书做概括训练）
   - 仿写（丁立梅仿写，每周2-3篇）
   - 造句仿写社群评比（每周最佳前三上榜）

3. **小学语文能力模型：**
   - 一年级：抓偏旁部首、字的结构（左中右/上中下/独体字）、口头问答
   - 二年级：字词+简单句式（永恒句式训练）、亲子共读、金波老师绘本
   - 三年级：开始用五维御学法、丁立梅开始刷修辞
   - 四年级：作文排比句训练、立意提升
   - 五年级：真正的坎（作文字数量要求更高、阅读题目更深更广、数学分数和方程）
   - 六年级：薅走作文精髓、开始抓初中文言文

4. **初中语文能力路径（按顺序）：**
   - 第一步：文言文（优先冲进度，听课→多读→实词虚词句式→一词多义→古今义→判断句→倒装句）
   - 第二步：古诗词
   - 第三步：阅读体系（按难度递进：非连续性文本→说明文→议论文→文学文本）
   - 第四步：作文积累（李娟、余秋雨、梁衡三位作家）
   - 持续：字词基础（天利38套，字音字形成语改错）

5. **语文园地使用：**
   - 每个单元开头有核心任务，语文园地是呼应这个任务的思路
   - 例：第一单元任务=概括+思想感情 → 园地就有交流平台讲思想感情
   - 例：第二单元任务=写读后感+古典名著 → 园地就有读后感和生僻词理解方法

**原文关键表述：**

> "周一到周五，语文最重要的事情是两个字：积累。没有比积累更重要了。"

> "我们小学抓课内是为了提升你的本质的能力。初中别用，初中没有那些东西，初中咱们直接干真题呢，没时间用。"

> "文言文是咱们的基础，你优先能够提前学的就这么一个东西。所以用好你的学习机，重点就是它的文言文部分。"

> "轻舟已过万重山。你旁边的人卷，就像两岸的猴子在叫，你真正要做的是把这艘船往前开。"

**案例：**
- 四年级同学作文案例："天空之所以如此的蔚蓝动人，是因为它包容了每一朵云彩，不论大小"（排比句仿写）

### 主题2：词汇/字词学习方法

**涉及文件：** 214855、222908、224914、225917、223911

**核心方法论：** 课内字词→造句→成语每日积累→五维御学法标蓝词

**具体做法：**

1. **课内字词造句法：** 每个单元翻开，选2-3个词，听一听、写一写、造个句。重点是让孩子把学的东西直接拿来造句。
2. **字词来源：** 考试里出的字词经常不是课本最核心的词，而是课文中出现但容易被忽略的边角词。
3. **五维御学法抓字词：** 每篇文章有蓝色标蓝的词，这些就是重点积累对象。
4. **听写改进法：** 不要只听写，要问"江河湖海的江是什么偏旁？""这四个字有什么相同特点？"——让大脑先反应。
5. **永恒句式训练：** 每天10分钟一篇，重点练句式。
6. **天利38套（初中）：** 字音字形成语改错，每天四道题，5-8分钟。"每一分钟一分"。

**原文关键表述：**

> "字词永远是拦路虎，你不去过那个关永远过不了。你中考过不了呢。"

> "语文字词考试里面经常出的不是课本里面的，你往往发现那些词汇好像就是你学过的，但是好像你从来都没有去写过。"

> "你不懂不要紧，你掌握方法就行了。"

**去重说明：** 课内字词造句在214855和224914重复，合并处理；永恒句式训练在223911和230921重复，以223911为基准。

### 主题3：造句训练与仿写

**涉及文件：** 214855、215859、222054、224914

**核心方法论：** 小切片练句法——作文不是让孩子直接创造，而是先借鉴再仿写再创造。

**具体做法：**

1. **小切片练句法（化整为零）：**
   - 一篇作文由很多段落组成，每个段落由小句子组成
   - 第一步练小句子，不练全篇
   - 小句子的组成 = 形容词 + 动词 + 成语 + 修辞手法

2. **丁立梅仿写流程：**
   - 每周2-3篇丁立梅文章
   - 读→找出修辞→理解为什么好→仿写（该替换的替换，该保留的保留）
   - 周末完成，提交给社群评比

3. **造句仿写社群机制：**
   - 每周评比，最佳前三上榜
   - 同年级横向比较+跨年级向上看

**原文关键表述：**

> "写作文是你孩子要去创造吗？还是借鉴？先去借鉴，借鉴好了之后变成自己的，然后就是输出，就可以创造了。"

> "你经常让孩子写作文的时候，孩子一脑懵，'妈我该写什么啊？'他大脑里面是一片空白，他甚至都没有看过这个东西该怎么写。"

> "重要的还是小切片。你无意之间给自己孩子点的那些技能点，突然打穿了，你的娃终于开窍了。"

**去重说明：** 小切片理论在214855和215859均有阐述，以215859更完整版为基准。

### 主题4：修辞积累（丁立梅体系）

**涉及文件：** 214855、215859、223057、224100、230105、230921、231025

**核心方法论：** 丁立梅这本书只学一样东西——修辞。主刷比喻，主刷拟人。一箭双雕：提升阅读理解+提升作文。

**具体做法：**

1. **适用年级：** 3-6年级，每周2-3篇。三年级就可以开始。

2. **核心刷法：**
   - 每篇文章找出所有修辞手法
   - 重点：比喻、拟人（其他修辞先不管）
   - 理解每个修辞为什么好、好在哪里
   - 仿写练习

3. **凌霄团队配套资源（13万字精批文档）：**
   - "由清华学习纯牛马凌霄亲自维护"
   - 每篇文章的修辞全部列出：用了什么修辞、重点是什么、为什么写得好
   - 例："小鱼一红一白两朵小关子"——比喻修辞，将金鱼比作小花，"通过在水里开三个字，生动形象的描绘出了金鱼在水中游动的画面"
   - 暑期已带领规划营孩子完完整整刷完13万字

4. **丁立梅仿写示例：**
   - 原句："池中的金鱼缓缓移动，像水中的花慢慢绽开"
   - 仿写方向：替换主体，保留句式和修辞结构

5. **阿婆坡上的丁立梅课程：** 小明老师已经录制好，每句修辞的要点全部讲清楚，仿写内容也提示好了，直接去仿写提交即可。

**原文关键表述：**

> "这个东西一箭双雕，一石二鸟。第一你读懂修辞为什么好，阅读理解能做很多题；第二你感受到它的美，作文也会有提升。"

### 主题5：文言文/古诗词学习路径

**涉及文件：** 221904、222908、215859

**核心方法论：** 文言文是初中语文唯一可以冲进度、提前学的内容。学习顺序：听课→多读→核心字词→句式。

**具体做法：**

1. **学习顺序（三步）：**
   - 第一步：听课（学习机上的文言文讲解，先入门）
   - 第二步：多读（晨读晚读大量去读，"熟能生巧"，先别背）
   - 第三步：核心知识点（实词、虚词、句式、一词多义、古今异义、判断句、倒装句）

2. **工具选择：** 文言文"一本"即可，不需要买很多本。学习机的语文同步功能先去用起来。

3. **时间安排：** 周末抽出30-40分钟。文言文和古诗词先搞定，阅读体系后攻克。

4. **古诗词：** 文言文学完后学古诗词。

5. **小学提前学：** 六年级就应该开始抓初中文言文。

**原文关键表述：**

> "文言文是咱们的基础，你优先能够提前学的就这么一个东西。"

> "家长就是喜欢文言文都还没开始学，就让他背。学了一个知识点就想让他做题。天呐，这可是初中。先别去背，先去读，熟能生巧。"

> "买了很多文言文一本通？文言文就是需要一本就行了，那一本书就叫做一本。"

**去重说明：** 221904和222908内容高度重叠，以221904的完整三步法+后续阅读体系路径为基准。

### 主题6：阅读理解（精读体系）

**涉及文件：** 224914、223911、222908、230105、230921

**核心方法论：** 阅读理解十道题里六道跟概括有关。核心能力 = 概括 + 找原文 + 人物形象分析。

**具体做法：**

1. **概括训练法（课本驱动）：**
   - 利用课本每篇课文后面的提示语
   - 问孩子两个核心问题：①"本文写了几件事？讲给老妈听一听。"②"本文抒发了作者什么样的思想感情？从哪儿看出来的？"
   - 关键是第二问的追问："从哪儿看出来的？"——考察找原文能力

2. **概括对照法（表格法）：**
   - 画表：左边是孩子自己概括的内容，右边是教辅书/参考答案
   - 对照左右差异，找缺什么
   - 不建议每天做，放在周末

3. **找原文方法论：**
   - 做阅读理解第一步 = 找答案、找原文
   - "你原文都找不到，妈你在说啥呢？你跟她说北京，她在跟你说南京。"

4. **人物形象分析：**
   - 典型题型："XX是一个什么样的人？"
   - 打法：找到原文描写（语言描写、动作描写等）→ 提炼特点

5. **初中阅读体系顺序：** 非连续性文本 → 说明文 → 议论文 → 文学文本（小说散文）

6. **初中名著阅读：**
   - 中考考5-8分，考的是细节不是情节
   - 小学提前读，但要有题目训练
   - 用万维名著导读

7. **精读课程：** 3-6年级阅读精读（凌霄亲自录），从题目分析到全文分析再到答案分析。

**原文关键表述：**

> "十道题里面起码有六道题左右都是跟概括相关的。"

> "做阅读理解，第一步叫做找答案、找原文，你得能把原文给找到才行。"

> "初中名著要在小学看完，看完也看不动。因为你看了之后也不会做题，没有题目的训练所有的阅读都没有太大的效果。中考题从来都不考情节太多，只考细节。"

**去重说明：** 概括方法在224914和222908均有详述，以224914的"课本驱动+表格对照法"为基准。

### 主题7：作文教学体系

**涉及文件：** 222054、225917、223057、232756、224914、231108

**核心方法论：** 作文从五个维度提升：词（形容词+动词+四字词语）→ 句（修辞：比喻排比）→ 段 → 框架（完整不头重脚轻）→ 立意（同龄人写不出的东西）。

**具体做法：**

1. **五维提升法：**
   - 第一维：词——动词、形容词、四字词语用得多不多？
   - 第二维：句——修辞手法（比喻、排比）用得多不多？
   - 第三维：段——段落结构是否合理？
   - 第四维：框架——是否完整？有没有头重脚轻？结尾写得好不好？
   - 第五维：立意——能不能写出同龄人写不出来的东西？

2. **立意提升——"剪秋"案例：**
   - 不要写"秋天树叶变黄了"这种老师看腻了的
   - 换一种方式：桂花落在满地 → 满地是银河、金色的星空坠满地
   - "剪秋"——在地上捡起秋天
   - 桂花分金桂、丹桂、白桂、四季桂，可以拼成向日葵、装在瓶中留香
   - 还可以剪枫叶、剪梧桐——把美好东西留住

3. **学生优秀作文案例：**
   - 排比句案例（四年级）："天空之所以如此的蔚蓝动人，是因为它包容了每一朵云彩，不论大小。花园之所以如此万紫千红，是因为它包容了每一朵花，不论美丑。溪水之所以如此活泼清澈，是因为它包容了每一条鱼儿在她的胸怀间跳跃，无论多少。"
   - 化用古诗案例（五年级）："我是时间，我会走的很慢很慢，像新雨过后的蜗牛，空山新雨后，天气晚来秋。像步履蹒跚的老人。假如我是时间，我不会是朱自清笔下的匆匆，我会慢慢的走。"

4. **课程产品：**
   - 小学1-6年级同步作文（凌霄亲自录）
   - 初中作文：同步作文 + 中考满分作文精讲
   - 2024年作文课成功命中福建和四川中考真题

5. **初中作文积累方向：** 李娟、余秋雨、梁衡三位作家的文章。

**原文关键表述：**

> "作文拿不到高分，写的不行呗。就你的词和你的句都不行，然后你的框架和立意不行。"

> "老师都看腻了。每六年教完一个小学，她都看腻这种文章了。你写点高级的东西好不好？"

> "凌霄的语文考146分，我们去年做的作文课成功命中了今年的中考真题，两个省福建和四川。"

**去重说明：** 五维提升法在222054最完整；学生作文案例在225917和224914重复，以225917展示更完整为基准。

### 主题8：英语学习规划

**涉及文件：** 220045、221051、225103、232111、232756

**核心方法论：** 幼儿园开始刷分级阅读（RISE/红火箭/海尼曼），二年级开始系统原版（Power Up），单词+发音是核心。盖兆全《做孩子最好的英语学习规划师》是原版领域圣经。

**具体做法：**

1. **分级阅读使用方法（RISE）：**
   - 不能干刷/穷刷，一定要有讲解
   - 第一遍：看图听（不要盲听，看图建立对应关系）
   - 第二遍：自己读
   - 遇到重复出现的重点词汇，家长要讲解意思（点读、查字典、亲自讲都可以）
   - 例：RISE F级别中"hiss"重复出现，要告诉孩子hiss是什么意思

### 主题9：英语单词学习法

**涉及文件：** 225103、232111、224100、230105、230921

**核心方法论：** 小白鹅单词书 = 课内知识点逐个精讲；初中用上海考纲词汇；背单词不要靠听写，靠例句翻译。

**具体做法：**

1. **小白鹅单词讲解风格：**
   - 每个单词都有词根词缀分析：例strawberry = straw（稻草）+ berry（梅）
   - 衍生词扩展：蓝莓blueberry、黑莓blackberry
   - 复数形式规则复习
   - 例句带读：Do you like strawberries? Yes, I do.
   - 高级词汇讲解：remember doing vs remember to do

2. **上海考纲词汇（初中推荐）：**
   - "没有谁比这本书更好了，更值得我去录"
   - 1785个词，讲解极细
   - receive → 补充相关用法
   - 年份会变（24版/25版）但内容不太会变

3. **背单词方法改进：**
   - ❌ 不要听写（低效）
   - ✅ 把例句盖住，让孩子翻译给你听（考察的是认不认识）
   - ✅ 朗读时要抑扬顿挫、断句（"英语也要断句，别人老外也得呼吸"）

4. **读霸精讲：** 全文讲解，重点是读不是做题。带翻译+语法点提取（mean to do、tell lies、be afraid to等固定搭配）

**原文关键表述：**

> "你把例句盖住，让它翻译给你听，这个才是你要做的事情。你的阅读理解不是靠你听写明白没？阅读理解是你认不认识那个单词。"

> "一千种方法来检验你的孩子，但是我就怕他第一步都不敢迈出。比如小白鸥，你今天学到这一节，你就把这个单词念出来，例句念出来。"

**去重说明：** 小白鹅讲解示例在225103和224100均有展示，以225103（strawberry更完整）为基准；背单词方法在232111最完整。

### 主题10：数学学习规划

**涉及文件：** 221051、220901、091248、090241

**核心方法论：** 数学问题只分四类：计算、知识点、解题思路、做题习惯。用复盘表定期诊断+录像回放法纠正习惯。

**具体做法：**

1. **数学复盘四要素（每周一次）：**
   - ①计算：有没有计算问题？抓了那么久为什么没效果？是题做少了还是方法有问题？
   - ②基础知识点：概念说说不清楚？跟电脑死机一样卡住了？
   - ③解题思路：计算过关了知识点也懂，但绕了一个数字就理解不清楚？
   - ④做题习惯：有没有圈画关键词？做完题有没有立刻检查？

2. **计算检查法：**
   - 做完一道题立刻检查一道题（不要从头到尾检查）
   - 每道题花3-5秒：有没有写漏、抄错、进退位
   - 30道题 × 5秒 = 150秒 = 2分半

3. **计算录像回放法（核心方法）：**
   - 用手机/支架录孩子做题过程（对准笔尖和题目，不要怼脸拍）
   - 回放给孩子看，让他自己发现问题
   - "运动员经常回放自己训练过程——分析对手打法，看自己哪些球没接好"
   - 家长不要当唐僧念经，让孩子自己看错误
   - 应用题同理：录下来看有没有圈画关键词

4. **计算能力提升：**
   - 大班/幼小衔接：洋葱计算入门+计算衔接
   - 小学核心：先抓计算，再抓错题
   - 巧算引入：用有趣的东西（巧算）解决不爱动脑的问题
   - 不爱动脑的突破口就是巧算

5. **分数段诊断：**
   - 95分：先搞到98再谈100，95和98有区别
   - 91分：一定出现了知识点或解题思路问题，"指哪打哪"
   - 80多分：计算+解题思路都有问题，可能公式都背不了
   - 90分以上：可以学高斯导引三星

6. **五年级是最大坎：** 方程+分数 → 应用题难度飙升

**原文关键表述：**

> "你的数学就这四个问题，你每周复个盘，看一看哪些问题真实存在，看一看比上一周又有了进步。"

> "你有方法去展现他自己错误的那一面。如果说有用的话，咱们就不会嫌唐僧唠叨。说有用的话，每个孩子都能考高分了。"

> "家长你怎么回事啊？这种谎话你还不会编吗？就跟老师商量好嘛。怎么那么埋头拉车不看路，辛辛苦苦犯错误呢？"

**去重说明：** 复盘四要素在220901和091248均有详述，以091248（最详细的使用说明）为基准；计算检查法在221051和220901重复，合并。
