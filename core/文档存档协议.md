# 文档存档协议

> **创建**：2026-05-02 | **版本**：v1.0

---

## 存档规则

### 1. 锋哥发来的内容 → `core/inbox/`

每次锋哥通过飞书发送文件（docx、图片、语音转文字等），Hermes在本地保存一份md版本：

- **文件名格式**：`inbox/YYYY-MM-DD_来源_主题.md`
- **来源标签**：飞书、语音、图片、截图
- **包含**：元数据（来源、日期、原始文件路径）+ 内容摘要/全文

### 2. Hermes生成的文档 → `core/`

Hermes主动生成的结构化文档直接放在 `core/` 下：

- 例如：`认知清空指南.md`、`记忆恢复指南.md`、`文档存档协议.md`
- 同时尝试同步到飞书云文档（需bot有docx:document:write权限）

### 3. 同步飞书云文档

- 创建文档：通过飞书API `POST /docx/v1/documents`
- 写入内容：通过飞书API `POST /docx/v1/documents/{id}/blocks/{id}/children`
- **前提**：bot需要有 `docx:document:write` 权限
- **状态**：2026-05-02 待授权

### 4. 记忆备份

- 每天自动备份记忆db到 `core/backups/memory/`
- 见 `core/记忆恢复指南.md`
