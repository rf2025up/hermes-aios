## 🔴 飞书断连事故（16:47-20:05，约2.5小时）
- **起因**：禁用旧feishu插件→PowerShell改json加BOM→openclaw-lark被设为disabled
- **排查过程**：
  1. 移除BOM → 没解决
  2. 恢复旧feishu enabled=true → 没解决
  3. npx doctor发现插件目录missing → 手动安装openclaw-lark → 没解决
  4. 删掉旧临时目录 → 没解决
  5. info --all发现 openclaw-lark.enabled=false → **这才是根因**
  6. openclaw config set plugins.entries.openclaw-lark.enabled true → 解决
- **根因**：openclaw-lark插件被设为disabled，飞书channel无法建立
- **⚠️ 教训**：
  1. 永远只用 `openclaw config` 命令改配置，绝不直接PowerShell改json
  2. PowerShell Set-Content会加UTF-8 BOM
  3. ConvertTo-Json可能丢失嵌套配置
  4. 出问题先跑 `npx @larksuite/openclaw-lark doctor` 和 `info --all`
  5. 已保存官方排错文档到 references/飞书官方插件指南.md
