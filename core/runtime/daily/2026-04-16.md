**Google AI Studio 对话批量导出**

### 前提条件（已就绪）
- Chrome 已安装（google-chrome-stable）
- 代理已通（Mac 192.168.1.105:6789）
- Chrome 已用代理启动，用户已登录 Google AI Studio
- 已安装 Google AI Studio Exporter 扩展

### 操作流程
1. 重启 gateway → 浏览器工具恢复
2. 用 browser 工具连接 Chrome（profile=user, target=host）
3. 在 Google AI Studio 对话列表页面
4. 逐个对话：
   a. 点击对话进入
   b. 滚动到对话最顶部
   c. 点击 "Export Now" 按钮（扩展按钮）
   d. 等待导出完成
   e. 返回对话列表，处理下一个
5. 所有导出的文件保存到 ~/google-aistudio-exports/
6. 生成索引文件 INDEX.md

### 注意事项
- Chrome 启动参数：--no-sandbox --proxy-server="http://192.168.1.105:6789"
- 大约70-100个对话，需要自动化循环
- 导出完成后清理临时文件
