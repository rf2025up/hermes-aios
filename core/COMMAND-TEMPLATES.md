# 高频命令模板（COMMAND-TEMPLATES.md）

> 沉淀高频语音命令，减少使用摩擦。用真实会说的话。

---

## 记录型

| 命令 | 示例 |
|------|------|
| 记账 | "买了53块钱的菜" / "发了3000工资" |
| 打卡 | "英语读完了" / "今天运动了" |

## 助理型

| 命令 | 示例 |
|------|------|
| 提醒 | "14:21提醒我XX" |
| 汇总 | "帮我捋一下进度" |
| 追踪 | "今天该干什么？" |

## 教练型

| 命令 | 示例 |
|------|------|
| 商业教练 | "帮我分析一下这个招生方案" |
| 复盘教练 | "帮我复盘一下今天的周会" |
| 沟通教练 | "我要跟家长打个电话，帮我设计话术" |
| 学习教练 | "帮我理解一下XX概念" |

## 执行型

| 命令 | 示例 |
|------|------|
| 整理文档 | "帮我整理一下这个周会纪要" |
| 生成反馈 | "帮我写个家长沟通纪要" |
| 排菜 | "帮我排下周的菜" |
