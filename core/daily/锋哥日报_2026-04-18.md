## 如何解决

### 技术方案优化
1. **智能滚动加载算法**：
   ```python
   # 检测可滚动元素，优先使用容器滚动
   scroller = document.scrollingElement or document.documentElement
   for (const el of all) {
       if ((style.overflowY === 'auto' || style.overflowY === 'scroll')) {
           scroller = el; // 找到真正的滚动容器
       }
   }
   ```

2. **渐进式检测机制**：
   - 检测滚动高度变化，连续5次稳定后认为加载完成
   - 最多滚动120次，避免无限循环

3. **文本智能过滤**：
   - 建立skip_prefixes列表，过滤UI元素文本
   - 保留有效对话内容，排除控制面板文本

### 流程改进
1. **小批量测试策略**：
   - 先测试3个对话验证方案可行性
   - 成功后再扩展到50个/批次

2. **多版本导出管理**：
   - exports: 早期版本测试
   - exports-v2: 优化版本
   - exports-v3: 最新CDP版本

3. **进度跟踪机制**：
